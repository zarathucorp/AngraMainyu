## 서문
AngraMainyu — 기존 질서를 흔드는 파괴적 정신.  
우리는 이 이름을 차라투의 창조적 에너지로 전환하여,  
기존의 파괴를 넘어 새로운 질서와 아름다움을 세운다.

## 목적
차라투스트라의 가르침을 기반으로,  
차라투 공동체의 사상과 철학을 신학적 원리로 정리하고,  
이를 의례적 승인과 기록 체계를 통해 제도화하여,  
공동체가 사상의 혈통을 이어받고 창조적 성장에 참여하도록 설계된 리포지토리입니다.

우리는 모두 **홍익인간 세계관에서 활약하는 아티스트**입니다.  
각자의 개성을 발휘하여 아름다움을 창조, 세상을 더 아름답게 만듭니다.

### 로고
<img src="img/Zarathu_Pattern.webp" width="300">

우리의 로고는 **1무늬 비대칭 타일링(Asymmetric Tiling)** 에서 영감을 얻었습니다.  
같은 기본 무늬를 반복하지만 결코 대칭이 나타나지 않는 패턴은, 차라투의 철학을 완벽히 표현합니다.

- **하나의 무늬**: 우리는 모두 인간이고 홍익인간을 꿈꿉니다.  
- **비대칭성**: 우리는 각자 자신만의 개성을 표현합니다.  
- **끝없는 창조**: 같은 인간이 서로 다른 개성을 표현, 세상이 빈틈없이 채워지고 끝없이 창조됩니다.  

차라투는 자신만의 개성을 가진 아티스트들이 모여 끝없이 아름다운 세상을 만들어갑니다.



## 경전
이 리포지토리의 경전은 니체의 **차라투스트라는 이렇게 말했다**입니다.  
차라투 공동체의 사상과 철학은 이 책의 가르침을 기반으로 발전하며,  
의례적 승인(Milestone)과 기록 체계를 통해 제도화되고 계승됩니다.

## 구조
- `ritual/` : 창조 서약, 구체화 절차, 봉헌 의례  
- `ideas/` : 경전 구절, 사상, 실행 지침  
- `contributors.md` : 모든 기여자 기록  
- `CHANGELOG.md` : milestone release 및 사상 기록 변화  

## 운영 원칙
1. 아이디어/경전 구절 작성 → PR 제출 → 공동체 검토  
2. 승인된 PR → Merge → milestone Release  
3. milestone Release → contributors.md 갱신 → 의례적 승인  
4. 모든 기록은 공개, 자유로운 사상 계승 가능
