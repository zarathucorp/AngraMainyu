# 차라투스트라 구절 + 실행 지침

## 공동체적 창조
- “너희는 나에게서 떨어져 나와 스스로가 되는 자가 되어라.”
- 실행: 새로운 아이디어를 작성하고 PR 제출 → 공동체 검토

## 의례적 승인
- “모든 위대한 일은 의식 속에서 시작된다.”
- 실행: milestone Release 시 의례적 공개

## 혈통과 기록
- “나의 사상은 바람처럼 퍼져, 너희 안에서 새로운 뿌리를 내리리라.”
- 실행: contributors.md 갱신 → 사상 혈통 유지

## 아름다움과 질서
- “세상은 혼돈 속에서 아름다움을 찾는 자에게 길을 보여주리라.”
- 실행: oath.md 4항과 연결 → 창조 목표 설정

## 자유로운 접근
- “진리는 닫힌 문 안에 있지 않고, 모든 이에게 열려 있다.”
- 실행: 모든 ideas/ 공개 → 공동체 자유 계승
